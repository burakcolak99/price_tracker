# RAM Fiyat Takip Botu

6 site üzerinden RAM kategorisini takip eden, fiyat düşüşlerini Telegram ile bildiren bot.

## Desteklenen Siteler
- Trendyol
- Hepsiburada
- Amazon TR
- Vatanbilgisayar
- İncehesap
- İtopya

## Kurulum
```bash
pip install -r requirements.txt
```

## Kullanım
```bash
python main.py
```

## Environment Variables
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
